■ 두께 건전성 분석표
배관 두께 분석 ASME B31.3 · API 570 기준
두께 비교 공칭 · 최소허용 · 측정최소
※ 압력설계두께 t = P·D ÷ (2·(S·E + P·Y))  [mm]  (ASME B31.3 §304.1.1)
※ 최소허용두께 tmin = t + CA  (부식여유 포함, 구조최소두께와 비교하여 큰 값)
※ 부식속도 CR = (Ti − t측정최소) ÷ 사용연수  [mm/y]  (API 570 §7.1.2 장기부식률)
※ 잔존수명 RL = (t측정최소 − tmin) ÷ CR  [year]  (API 570 §7.2)
※ 최대사용압력 MAWP = 2·S·E·t측정최소 ÷ D  [MPa]  ·  Y=0.4(강, T<482℃)
